\subsubsection*{template$<$class T$>$\newline
class Pila$<$ T $>$}

T.\+D.\+A. \mbox{\hyperlink{classPila}{Pila}}. 

Una instancia {\itshape v} del tipo de datos abstracto \mbox{\hyperlink{classPila}{Pila}} sobre el tipo {\ttfamily T} es una lista de elementos del mismo con un funcionamiento {\itshape L\+I\+FO} (Last In, First Out). En una pila, las operaciones de inserci{\ucr}n y borrado de elementos tienen lugar en uno de los extremos denominado {\itshape Tope}. Una pila de longitud {\itshape n} la denotamos


\begin{DoxyItemize}
\item \mbox{[}a1,a2,a3,..,an$>$
\end{DoxyItemize}

donde {\itshape ai} es el elemento de la posici{\ucr}n i-\/{\ucr}sima.

En esta pila, tendremos acceso {\ucr}nicamente al elemento del {\itshape Tope}, es decir, a {\itshape an}. El borrado o consulta de un elemento ser{\ucr} sobre {\itshape an}, y la inserci{\ucr}n de un nuevo elemento se har{\ucr} sobre {\ucr}ste. Quedando el elemento insertado como {\itshape Tope} de la pila.

Si {\itshape n=0} diremos que la pila est{\ucr} vac{\ucr}a.

El espacio requerido para el almacenamiento es O(n). Donde n es el n{\ucr}mero de elementos de la \mbox{\hyperlink{classPila}{Pila}}.
